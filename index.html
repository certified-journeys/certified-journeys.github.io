About the Project
certified-journeys is a free, open-source learning infrastructure for senior engineers who learn in public.

No login. No cost.

Every tracker is a single HTML file. Progress lives in your browser via localStorage. No server, no account, no upsell — ever.

Built to be forked.

Fork any repo, edit two data arrays, deploy to GitHub Pages. Your own cert tracker is live in under an hour.

Open by design.

No tracking, no analytics, no ads, no premium tier. If your journey helped you pass — share it, improve it, and pay it forward.

How to fork your own tracker

  • Fork any repo at github.com/certified-journeys
  • Open index.html — find days and topics arrays
  • Replace the data with your cert's study plan
  • Settings → Pages → main branch → Save
  • Live at yourname.github.io/repo-name in 2 min
  • Username and storage key auto-detected from URL

Tech decisions

  • Single HTML file — no build tools, no npm, no bundler
  • localStorage only — data stays on your device
  • MIT licensed — fork, adapt, remix freely
  • No external dependencies — works offline after first load
  • Dark mode via prefers-color-scheme
  • Responsive — mobile, tablet, desktop
