## Why I built certified-journeys

Every certification resource I found had the same problems:

- Paywalled behind courses I didn't need at senior level
- Built for complete beginners — no senior context, no nuance
- Scattered across 10 different tabs with no way to track progress
- Required an account, an email, or a subscription just to get started

So I built what I actually needed — a structured, trackable, senior-focused daily plan that runs entirely in the browser, saves locally, costs nothing, and gets out of the way.

The first tracker was for PSPO I. The format worked so well I open-sourced it as a template so anyone can fork it and build their own for any certification.

## Contributing

All contributions are welcome — corrections, better tips, additional quiz questions, or entirely new cert trackers built from the template.

**To fix something in an existing tracker:**
1. Fork the repo
2. Edit `index.html` or `README.md`
3. Open a pull request with a clear description

**To add a new tracker to the collection:**
1. Fork `pspo-certified` as your base
2. Follow the forking guide in the README to customise it for your cert
3. Open a PR to add your repo to the community table

The only hard rule: no tracking, no analytics, no paywalls, no external dependencies. Every tool in this collection must remain free and self-contained.
